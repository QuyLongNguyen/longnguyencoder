Hướng dẫn gửi mail với Mailjet API

13/02/2023

By Long Nguyen

Mailjet là một trong những dịch vụ gửi email và sms phổ biến, với trình quản lý email và các tính năng mạnh mẽ, Mailjet giúp cho việc gửi email trở nên đơn giản hơn bao giờ hết. Trong bài viết này, mình sẽ cùng thực hiện việc gửi mail với Mailjet.

1. Cách đăng ký Mailjet

Việc đăng ký Mailjet vô cùng đơn giản, bạn chỉ cần sử dụng một email là đủ. Mailjet không yêu cầu thông tin về số điện thoại hay thẻ tín dụng.

Để đăng ký, truy cập tới màn hình đăng tý tại đây. Nhập tên tài khoản là một email đã xác thực và mật khẩu muốn đặt cho tài khoản.

Sau đó, bấm tiếp tục và nhập những thông tin theo yêu cầu. Đồng thời, bạn có thể chọn gói dịch vụ sẽ sử dụng bằng cách chọn Change plan trong mục Your plan. Mặc định, Mailjet sẽ cung cấp gói miễn phí, cho phép gửi 200 emails/ngày tương đương 6000 emails/tháng.

Tiếp tục chọn Next cho tới khi điền thông tin hoàn tất . Lúc này, Mailjet sẽ gửi một confirmation tới hòm thư của email mà mình sử dụng để đăng ký tài khoản.

Trong hộp thư đến của hòm thư, tìm tới mail và Mailjet đã gửi, nhấn vào Active Your Account để kích hoạt tài khoản, hoàn tất quá trình đăng ký.

Sau khi nhấn kích hoạt, Mailjet sẽ tự động kích hoạt tài khoản và điều hướng về trang dashboard của mailjet. Lúc này, việc đăng ký tài khoản đã thành công.

2. Sử dụng dịch vụ gửi mail của Mailjet API

Để có thể sử dụng Mailject API, bước đầu tiên ta cần lấy được Api Key và Secret Key để ứng dụng của mình có thể gọi tới API. Để có thể lấy được key, tại mục Account Settings, chọn Api Key Managemnent.

Danh sách các Key sẽ được hiển thị trong API Managment, mặc định sẽ có một API Key tạo sẵn cho Primary Account. Để tạo Secret Key, chọn Generate Secret Key. Secret Key chỉ hiện một lần sau khi tạo, vì vậy hãy lưu lại, nếu không ta sẽ cần reset để tạo lại một Secret Key khác.

Sau khi tạo key thành công, lúc này ta có thể gọi tới Mailjet API trong ứng dụng của mình. Cách thức sử dụng Mailjet API được mô tả chi tiết trong API Document. Cụ thể là Send API đối với việc gửi mail. Ở trong phần tiếp theo, ta sẽ thực hiện gửi Mail bằng Mailjet trong ứng dụng Java.

API Document đã mô tả chi tiết cách gửi Mail với Code Sample của một số ngôn ngữ thông dụng, trong đó có Java như bên dưới: 

package com.my.project;
    import com.mailjet.client.errors.MailjetException;
    import com.mailjet.client.errors.MailjetSocketTimeoutException;
    import com.mailjet.client.MailjetClient;
    import com.mailjet.client.MailjetRequest;
    import com.mailjet.client.MailjetResponse;
    import com.mailjet.client.ClientOptions;
    import com.mailjet.client.resource.Emailv31;
    import org.json.JSONArray;
    import org.json.JSONObject;
    
    public class MyClass {
        /**
         * This call sends a message to one recipient.
         */
        public static void main(String[] args) throws MailjetException, MailjetSocketTimeoutException {
          MailjetClient client;
          MailjetRequest request;
          MailjetResponse response;
          client = new MailjetClient(System.getenv("MJ_APIKEY_PUBLIC"), System.getenv("MJ_APIKEY_PRIVATE"), new ClientOptions("v3.1"));
          request = new MailjetRequest(Emailv31.resource)
                .property(Emailv31.MESSAGES, new JSONArray()
                    .put(new JSONObject()
                        .put(Emailv31.Message.FROM, new JSONObject()
                            .put("Email", "pilot@mailjet.com")
                            .put("Name", "Mailjet Pilot"))
                        .put(Emailv31.Message.TO, new JSONArray()
                            .put(new JSONObject()
                                .put("Email", "passenger1@mailjet.com")
                                .put("Name", "passenger 1")))
                        .put(Emailv31.Message.SUBJECT, "Your email flight plan!")
                        .put(Emailv31.Message.TEXTPART, "Dear passenger 1, welcome to Mailjet! May the delivery force be with you!")
                        .put(Emailv31.Message.HTMLPART, "<h3>Dear passenger 1, welcome to <a href=\"https://www.mailjet.com/\">Mailjet</a>!</h3><br />May the delivery force be with you!")));
          response = client.post(request);
          System.out.println(response.getStatus());
          System.out.println(response.getData());
        }
    }

Khi send thành công, Mailjet sẽ trả về Status Code là 200, kèm theo Response Body như bên dưới:

Có thể thấy, cách gửi Mail với Mailjet về cơ bản là ta sẽ dựng lên một JsonObject trong Java và method put để thêm các thuộc tính dữ liệu cần thiết để tạo ra Json trong Request Body. Cách triển khai khiến code trở nên khó đọc, và khó mở rộng thêm. Vì vậy dưới đây, mình sẽ mô tả cách triển khai tối ưu hơn, ta sẽ tạo ra một Request DTO để chứa thông tin Mail Request dưới dạng DTO (Data Transfer Object) rồi mới chuyển tất cả sang JsonObject.

2.1 Tạo các class DTO

Tạo class EmailDto.java để chứa thông tin một Mailer, bao gồm: tên email và tên người dùng email.

@AllArgsConstructor
@Getter
@ToString
public class MailerDto {
    private String email;
    private String name;
}

Tạo class AttachmentDto.java chứa thông tin của file đính kèm, bao gồm: tên file sẽ hiển thị, dạng nội dụng và nội dung File được mã hoá ở dạng base64.

import org.example.enums.ContentType;

@NoArgsConstructor
@AllArgsConstructor
@Builder
public class AttachmentDto {
    private String fileName;
    private ContentType contentType;
    private String base64Content;
}

Ta sẽ khai báo enum ContentType.java, để định nghĩa các loại ContentType. @SerializedName được sử dụng để chỉ ra giá trị của Enum Field khi được Convert sang Json.

import com.google.gson.annotations.SerializedName;

public enum ContentType {
    @SerializedName("text/html")
    TEXT_HTML,
    @SerializedName("text/plain")
    TEXT_PLAIN,
    @SerializedName("text/csv")
    TEXT_CSV,
    @SerializedName("video/mp4")
    VIDEO_MP4,
    @SerializedName("image/gif")
    IMAGE_GIF,
    @SerializedName("image/jpeg")
    IMAGE_JPEG,
    @SerializedName("image/png")
    IMAGE_PNG;
}

Và cuối cùng, là class EmailDto, đại diện cho toàn bộ thông tin muốn gửi trong một Sending Mail Request.

Như vậy ta đã tạo xong các DTO cần thiết, tiếp theo sẽ cần một Email Service.

2.2 Tạo class Email Service

Define ra method gửi Mail mà một Email Service cần có. 

public interface EmailService {
     boolean sendEmail(EmailDto emailDto);
}

Tạo implementation của EmailService, là class MailjetService. Các thông tin về nhạy cảm như Secret Key, nên để trong một file properties, tránh sử dụng hardcode. 

public class MailjetService implements EmailService {
    private final String apiKey = ""; //your api key
    private final String secretKey = ""; //your secret key
    private final String mailJetVersion = "v3.1"; //mailjet api version
        
    @Override
    public boolean sendEmail(EmailDto emailDto) {
        MailjetClient client = new MailjetClient(apiKey, secretKey, new ClientOptions(mailJetVersion));
        MailjetRequest request = new MailjetRequest(Emailv31.resource)
                .property(Emailv31.MESSAGES, new JSONArray()
                        .put(new JSONObject(new Gson().toJson(emailDto))));
        try{
            MailjetResponse response = client.post(request);
            System.out.println(response.getStatus());
            System.out.println(response.getData());
            return true;
        }catch (Exception e){
            return false;
        }
    }
}

Như vậy, ta đã tạo xong một Mail Service và sẵn sàng để gửi mail. Tiếp theo, ta sẽ khởi tạo một EmailDto và test trong hàm main.

Và cuối cùng, thực hiện khởi tạo EmailDto và gọi Mail Service để Send Mail.

public static void main(String[] args) throws IOException {
    EmailService emailService = new MailjetService();

    //Create host mail
    String hostEmail = "longnq.tech@gmail.com"; //your host email
    String hostName = "Long Nguyen Coder"; //your host name

    //Create "mail to" list
    MailerDto toMailer = new MailerDto("famailtraptest1@gmail.com", "FA Mailtrap"); //mail to
    List<MailerDto> toMailerList = Arrays.asList(toMailer);

    //Create mail body
    String mailBody = FileUtils.readFileToString(new File("welcome-mail.html"), "UTF-8");
        //Pasting dynamic content to mail template
    Map<String, String> mailAttributes = new HashMap<>();
    mailAttributes.put("name", toMailer.getName());
    StringSubstitutor stringSubstitutor = new StringSubstitutor(mailAttributes);
    mailBody = stringSubstitutor.replace(mailBody);

    //Create attachments
    File attachedFile = new File("attachment.txt");
    byte[] fileBytes = IOUtils.toByteArray(new FileInputStream(attachedFile));
    String base64content = Base64.getEncoder().encodeToString(fileBytes);
    AttachmentDto attachment = new AttachmentDto(attachedFile.getName(), ContentType.TEXT_PLAIN, base64content);
    List<AttachmentDto> attachments = Arrays.asList(attachment);

    //Create email dto
    EmailDto email = EmailDto.builder()
            .from(new MailerDto(hostEmail, hostName))
            .to(toMailerList)
            .subject("Mail demo greetings")
            .htmlPart(mailBody)
            .attachments(attachments)
            .build();

    //Send email
    System.out.println(emailService.sendEmail(email) ? "Send success" : "Send failed");
}
